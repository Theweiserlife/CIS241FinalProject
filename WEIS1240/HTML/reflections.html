Reflections

I feel like this class got off to a bumpy start with the course material not being available. I usually purchase my books ahead of time so that I have a good start on all my classes. I was running two weeks behind in this class from the beginning and never really felt like I was able to catch up. But I did pick up a thing or two along the way

My favorite thing I learned this quarter was stylesheets and how to apply them to a website. Before this class I was applying all of my CSS and Javascript per page which would end up being a lot of code. Now I know how to apply style to a whole website and make it look professional.
